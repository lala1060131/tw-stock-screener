📖 使用說明

🤖 這個工具做什麼？

自動從台股中篩選「低基期獲利股」——即財務健全、股價處於歷史低水位、且開始出現轉機訊號的股票。原理是利用三道篩選門層層過濾，找出最接近理想「低買」條件的標的。

🛡 三大篩選門詳解

第一門：財務防護門
  • 連5年EPS > 0：確保公司持續賺錢，不是靠業外美化財報
  • 負債比 < 50%：過濾財務槓桿過高、可能撐不到景氣回溫的公司
  • 有發放股利：等待期間能「領息等價差」，降低持有心理成本
第二門：低基期辨識門
  • PB < 1.2：股價接近公司帳面淨值，下檔風險有限
  • 10年位階 < 20%：公式 = (當前 - 10年低點) / (10年高點 - 低點)，數值越小代表股價越接近歷史低點
第三門：轉機訊號門
  • 站上季線（MA60）：低迷後突破60日均線，代表中長期趨勢開始轉強
  • 量能上升：近30天成交量 > 前30天，資金開始關注
  • RSI 分析：RSI 30~50 為超跌回升的黃金區間

📈 月營收 YoY

透過 FinMind API 取得台股月營收數據，計算年增率（YoY）。由負轉正是最強的轉機訊號之一——代表基本面開始改善。建議優先關注「通過2門＋營收由負轉正」的標的。

📊 回測功能

針對每支股票，找出過去5年內所有「MA60穿越買入訊號」，計算持有N天後的報酬率與勝率。建議選擇勝率 > 60%、平均報酬 > 5% 的標的。

💬 LINE 推播

需申請 LINE Notify Token。設定後，每次篩選完成自動推播結果到你的 LINE。也可在「設定」頁手動觸發推播。

⚠️ 免責聲明

本工具僅供學習與研究用途，所有篩選結果基於歷史數據自動計算，不構成任何投資建議。投資涉及風險，請投資人自行評估並承擔風險。
